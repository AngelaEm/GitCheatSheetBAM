git commit -m "[message]"

Kommandot används för att göra en "commit" till ett Git-repo. Den används för att spara ändringarna som man gjort till det lokala "commits" inför push.

git restore [file]

Kommandot används för att återställa filen i ditt Git-repo till dess senaste sparade tillstånd.

git push

Kommandot används för att skicka (pusha) dina lokala commits till en fjärrserver (remote repository).
