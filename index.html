git config --global user.name “[namn]”

Sätter ditt namn globalt i Git. Detta är något man gör för första gången.

git config --global user.email [epost adress]

Sätter din epost adress globalt i Git. Detta är något man gör för första gången.

git config --global color.ui.auto

Sätter Git färg palett till standardfärger.

git init

Skapa repository

git clone [url]

klonar en kopia av koden och dess historik till sin dator från tex github,Azure DevOps.

git branch [Branch-namn]

Skapa ny branch

git checkout [branch-namn]

Checka ut branch. Man flyttar filerna från masterbranschen till en bransch.

git merge [branch-name]

Slår samma masterbrachen med branchen.

git branch -d [branch-name]

Tar bort branchen.
