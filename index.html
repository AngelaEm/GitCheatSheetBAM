git gc

Står för "garbage collect". Rensar upp onödiga filer för att optimera det lokala git-repot. Git sparar en massa data när man arbetar och det kan bli ganska mycket. Efter ett tag är en del data inte längre nödvändig eller användbar och då kan man rensa med git gc. Kan vara lurigt att använda om man inte har full koll på vad som rensas bort, speciellt om man delar repo med andra.
